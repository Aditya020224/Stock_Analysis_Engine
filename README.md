# StockPulse AI 📈
A professional Deep Learning web application that predicts stock prices using LSTM (Long Short-Term Memory) networks.

## ✨ Features
* **Real-time Data:** Fetches live market data via Yahoo Finance API.
* **Deep Learning:** Built with a Sequential LSTM model for time-series forecasting.
* **Responsive UI:** Dark-themed dashboard built with Flask and Bootstrap.

## 🛠️ Tech Stack
* **Backend:** Python (Flask)
* **AI/ML:** TensorFlow, Keras, Scikit-learn
* **Data:** Pandas, Numpy, YFinance
* **Deployment:** GitHub, Render

## 📋 How to Run Locally
1. Clone the repo: `git clone [YOUR_REPO_LINK]`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`
